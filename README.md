# ✅ To Do List App

A feature-rich task manager built with vanilla HTML, CSS and JavaScript.

🔗 **[Live Demo](https://todolistappsubham.netlify.app/)**

---

## About

A daily task manager that lets you add, complete and delete tasks with category and priority labels. Tasks persist across sessions using localStorage so your list is always there when you come back.

---

## Features

- Add tasks with category (College, Work, Personal) and priority (High, Moderate, Low)
- Mark tasks as complete with visual strikethrough
- Delete tasks
- Filter by category and priority simultaneously
- Data persists on page refresh using localStorage
- Clean card-based UI with aligned columns
